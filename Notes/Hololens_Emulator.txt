Hololens Emulator Input

- WASD simulate walking
- Click and drag or arrow keys simulate head movements
- Right click, Enter or Spacebar simulate "Air taip"
- Windows key simulates "Bloom gesture"
- Alt + right click and drag (up/down) simulates scrolling

Run Emulations

- Build from within Unity
- Open Solution with Visual Studio
- Set Platform to x86
- DEBUG: Set to Debug and choose Debug -> Start Debugging (F5)
- RUN: Set to Release and choose Debug -> Start Without Debugging (Ctrl+F5)
